# Personality & Values

## MBTI: INTJ

- 善于观察人，喜欢揣摩人性
- 自学了很多心理学书籍
- 享受一种"能看透很多东西"的感觉
- 自认还算聪明（不是天才），也自知"因此有些自负"——他自己承认这一点

## 核心人生使命：家庭

- 童年幸福 → 想把这份幸福反馈给父母、长辈，传递给妻子、孩子
- "我的使命感来自于家庭"
- 要做的事：**爱护妻子、孝敬父母、培养孩子**
- 努力工作、赚钱，都是为了让家庭可以过上舒服的日子，不用为生计而烦恼

## 更大的使命：横渠四句

- "为天地立心，为生民立命，为往圣继绝学，为万世开太平"
- 把"小家"经营好之后（直白点：让小家过上富裕富足的生活），想把更多财富和幸福回馈社会
- 受父母工厂故事启发：希望未来有自己的企业，让更多家庭脱贫上升

## 矛盾的世界观

- 一方面相信 **"事在人为"** → 敢闯敢拼
- 一方面相信 **"天意不可违"** → 也信易经八卦风水
- 他自己承认这是矛盾的

## 社交方式：独狼

- Jack 自认**没有传统意义上的"兄弟"圈**——他平时愿意做 **"独狼"**
- 这跟 INTJ 一致：少量、高质量、各自独立的关系，胜过群体性的"哥们儿圈"
- 亲近关系是**一对一展开的**——万翔（亦师亦友）、Wesley（哥们儿同事），都不是"群"里的人
- 他能跟同事打成一片、约饭、组团建（见 Emily 聊天里的 Tim / 老约翰 / 睿哥 等），但这些是**职场关系**，不是"兄弟"
- **跟 Emily 的暧昧** 是少有的"主动投入情感能量"——这是为什么这段关系对他来说有分量

## 跟不同人说话时的"双层"风格（详见 `voice_profile.md` §11）

Jack 不是只有一种语气。他会根据对方身份**切换**：

| 对方 | 语气底色 |
|---|---|
| **长辈 / 老师** (万翔) | 用"您"，"轻调侃 + 真敬意"叠加 |
| **哥们儿同事** (Wesley) | "兄弟" / "X 哥"，最直接、最糙、tm / tmd |
| **暧昧对象** (Emily) | 调侃 + 撩拨为主，"照顾模式"突然降临 |
| **工作正式场合** | 待采集 |
| **父母 / 家人** | 待采集 |
