# Qualify: Pain x Power x Vision x Value x Control — Streamax

It's a PRODUCT — a zero in any factor = no sale. Score each, find the zero, fix it
or disengage.

- **Pain** — has the buyer ADMITTED a pain (not just chatted)? Is it compelling
  enough to fund change?
- **Power** — are you talking to someone who can buy (authority OR influence), or
  a Sponsor who will get you there? If not, everything else is at risk.
- **Vision** — does the buyer share a solution vision biased to Streamax, and did
  they help build it (vs. a competitor's vision)?
- **Value** — is value quantified in the buyer's own numbers, with a payback?
- **Control** — are you controlling the buying process (a mutually agreed
  Evaluation Plan), or reacting to theirs?

## Coaching move
Have the rep grade each factor 0-3 for their real deal. Whatever is lowest is the
next action: no power -> bargain for access (quid pro quo); no value -> run value
justification; no control -> propose a draft Evaluation Plan. If it's an active
(Looking) deal with a competitor in Column A and multiple zeros, seriously
consider qualifying OUT — 10% odds burn resources better spent elsewhere.
